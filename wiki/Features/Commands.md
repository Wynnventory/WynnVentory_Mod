# Commands

## Prefix
- All Wynnventory client-side commands begin with: `/wynnventory`

## Available commands
- `/wynnventory send`
  - Immediately flushes any queued crowdsourced data (lootrun pools, raid pools, trade market listings, gambit items) to the Wynnventory API.
  - Prints a chat confirmation message when done.

## Tab completion
- Typing `/wynnventory` shows subcommand suggestions in the client.

## Notes
- These are client-only convenience commands implemented via Brigadier and do not require server permissions.
