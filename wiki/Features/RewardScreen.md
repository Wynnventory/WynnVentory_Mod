# Reward Screen

## Open the Reward Screen
- Default keybind: N (can be changed under Controls > Key Binds > Wynnventory)

## What it shows
- Consolidated view of all tracked reward pools (Raid, Lootrun, and other reward sources supported by Wynnventory).
- Each pool card shows:
  - Short name (e.g., NOL, TNA, etc.)
  - Items in the pool (rarity-colored)
  - Mythic Aspects for raid pools (see Aspect Overlay)

## Filters and search
- Toggle item rarities using the filter buttons (per-user saved in `config/wynnventory.json`).
- Rarity toggles:
  - Mythic, Fabled, Legendary, Rare, Unique, Common, Set, and Unusable
- Search box filters items by name (Wynntils original item names).

## Settings reference (RewardScreenSettings)
- showMythic
- showFabled
- showLegendary
- showRare
- showUnique
- showCommon
- showSet
- showUnusable

## Notes
- Favourite items (from Wynntils Favourites) will also trigger notifications when present; see Favourite Notifications.
