# Keybinds

## Category
- All keybinds are grouped under the category: Wynnventory

## Keybinds
- Open Reward Screen
  - Translation key: `key.wynnventory.openRewardScreen`
  - Default: N
  - Allowed in inventory: No
  - Action: Opens the Wynnventory Reward Screen
- Toggle Tooltips
  - Translation key: `key.wynnventory.toggleTooltips`
  - Default: Unassigned
  - Allowed in inventory: Yes
  - Action: Toggles TooltipSettings.showTooltips and prints a chat message
- Toggle Boxed Tooltips
  - Translation key: `key.wynnventory.toggleBoxedTooltips`
  - Default: Unassigned
  - Allowed in inventory: Yes
  - Action: Toggles `showBoxedItemTooltips`. When disabled, no secondary tooltip will be shown for boxed items.

## Notes
- Keybinds are disabled while typing in chat or in a focused text box to prevent accidental triggers.
- You can search for “Wynnventory” in Controls > Key Binds to locate them.
